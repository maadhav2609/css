      Q]How to embed audio and video in a webpage?   
      Ans: The HTML5 < audio > and < video > tags make it simple to add media to a website. You need to set src attribute to identify the media source and include a controls attribute so the user can play and pause the media.

      Q] Semantic element in HTML5?  
      Ans:A semantic element clearly describes its meaning to both the browser and the developer. Examples of non-semantic elements: < div > and < span > - Tells nothing
